1. Introduction

The document contains specific information with the regular classes and functionality about the design of the application. The aim of this project is to digitize the existing documents/files via scanning. The converted digital format will be stored into storage using a simple document system capable search and retrieval for attaching the same. The project is a government-based initiative. The project has a user management system while controlling their own and public documents in a digitalized form. Users have different ranking and control over the documents accordingly. This document digitization service will help employees and users to get information in a faster way. Being user friendly and efficient, this application will benefit the organization. this documentation system will provide a substantial service in further business strategies. It will help the Admin and the User to access the documentation easily and they can store those documents as per needed.

4.High level Component Design

User:

Every user has one or more specific roles for their account. They can access the file they have uploaded which is only viewed by them in their specific account, access files according to their role and access all public files that are published. Google login, password recovery via admin, avatar for profile will be a feature.

Authentication:

Account authentication is secured by the user. Users can recover password, use google login. Only administrator level users will be able to create account, edit roles, delete accounts while also retaining highest class of editing access across all user files.

User Management:

User profile will have permission to edit their own uploaded documents and view public files. Files provided for specific roles can also be accessed. Only administrator level user will have permission for all access to all user files and user account modifications.

Document Management:

Public and private repositories for document management. There are both public files for each user which will be accessible via roles and private files will be accessible to uploading user only. Document indexing, document archiving will be the features.

Notification Management:

Files shared for public access and role specific access will be shown in an accessible notification tile which when clicked will show the uploaded document.

Search:

Files can be searched via file name, indexing, upload date, uploader. Only accessible files will be shown in the search according to the user. Full freedom for searching across all available documents is only available for the administrator.
